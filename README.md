# AdventOfCode 2018

Solve [Advent of Code 2018](https://adventofcode.com/2018) puzzles with [elixir](https://elixir-lang.org/).

## Installation

```
mix deps.get
mix escript.build
```

## Usage

To solve second part of a puzzle from day 1 run

```
cat input/day1.txt | ./advent --day 1 --task 2
```
